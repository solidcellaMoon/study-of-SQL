# 4. IS NULL

* [문제 리스트 링크](https://programmers.co.kr/learn/courses/30/parts/17045)



## 이름이 없는 동물의 아이디

```mysql
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NULL
```



## 이름이 있는 동물의 아이디

```mysql
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID
```



## NULL 처리하기

```mysql
SELECT ANIMAL_TYPE, IFNULL(NAME, 'No name') AS NAME, SEX_UPON_INTAKE
FROM ANIMAL_INS
```
